Status colors:
	* White: 255, 255, 255
	* Yellow: 255, 202, 7
	* Blue: 0, 38, 184
	* Green: 52, 167, 49
	* Orange: 255, 113, 27
	* Red: 220, 33, 49
	* Gray: 102, 102, 102

Status meaning:
	* White: Scheduled
	* Yellow: Boarding
	* Blue: In Flight
	* Green: Landed
	* Orange: Delayed
	* Red: Cancelled
	* Gray: Gate closed
	* Flashing yellow: Final Call
		
GateRecord Statuses:
	* 0 = Inactive
	* 1 = Available
	* 2 = Occupied

TerminalRecord Statuses:
	* 0 = Inactive
	* 1 = Available

Crew Statuses:
	* 1 = Available
	* 2 = Not Available

Aircraft Statuses:
	* 0 = Inactive
	* 1 = Just added in db
	* 2 = Crews added
	* 3 = Route assigned
	* 5 = Scheduled
	* 6 = Boarding
	* 7 = In Flight
	* 8 = Landed
	* 9 = Delayed
	* 10 = Cancelled
